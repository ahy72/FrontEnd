# フロントエンド側

## http://systrialsvr/VirtualMachineStatus2/Machines

* 仮想マシンの稼働状況一覧を表示する
* [更新] ボタンを押すと、[状態] と [接続] が更新される

## http://systrialsvr/VirtualMachineStatus2/Message

* 稼働状況一覧に表示するメッセージを設定する
* 「メンテナンス中」のような用途を想定している
